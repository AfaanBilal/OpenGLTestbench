OpenGL Testbench
================

Author: [Afaan Bilal](https://afaan.dev)

A series of graphics tests implemented in C++ using OpenGL (GLEW), GLFW, GLM, Dear ImGui and STB.

## Testbench tests:

- Clear color
- Basic Triangle
- Basic Quad
- Basic Texture 2D
- Raytracing (CPU and GPU)
